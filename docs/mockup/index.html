Seuraavat askeleet
Konkreettinen etenemissuunnitelma konseptista pilottiin
1

Sidosryhmähaastattelut

Valitaan 3-5 pientoimittajaa ja kaupan edustajat. Haastatellaan tarpeet transcript päällä — AI tiivistää vaatimukset automaattisesti.

Viikot 1-2
2

Konseptin validointi

Esitellään tämä mockup + konseptidokumentti asiakkaalle. Sovitaan pilottimyymälät ja -toimittajat sekä kassadatan syöttötapa.

Viikko 2-3
3

Tekninen setup

AWS-tili + Bedrock. WhatsApp Business API rekisteröinti. n8n asennus. Bedrock Agent luonti. Ensimmäinen toimittaja testaa.

Viikot 3-4
4

Pilotti — 5-10 toimittajaa

Onboarding WhatsAppiin. Manuaalidata syötetty. Kaupan henkilöstö käyttää n8n:ää. Kerätään palaute ja iteroidaan.

Kuukaudet 1-3
5

Tuotanto — oikeat integraatiot

SAP BTP / WMS integraatio. Web-portaali toimittajille. Confluent Kafka real-time eventsille. Laajennus muihin alueisiin.

Kuukausi 4+
